Credit Risk Assessment

This project focuses on developing a machine learning classification model to evaluate the creditworthiness of loan applicants using historical financial data. The model analyzes features such as income, credit history, loan amount, and repayment behavior to predict the likelihood of default.

The workflow includes data preprocessing, handling missing values, feature selection, and training classification algorithms such as Logistic Regression, Decision Trees, and Random Forest. Model performance is assessed using metrics like accuracy, precision, recall, and ROC-AUC to ensure reliable risk prediction.

Tech Stack

Python, Pandas, NumPy, Scikit-learn, Matplotlib

Objective

To demonstrate the application of machine learning techniques in financial risk assessment and support data-driven lending decisions.
