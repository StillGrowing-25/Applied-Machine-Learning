Overview
This project focuses on predicting stock prices using a time series forecasting approach. Unlike traditional machine learning models, this project uses Long Short-Term Memory (LSTM) networks to capture temporal dependencies in stock price data.

Objective
To build a deep learning model that can analyze historical stock price data and forecast future closing prices.

Dataset
The dataset used in this project is the Tesla Stock Price Dataset obtained from Kaggle. It contains historical stock data including:

-Date
-Open
-High
-Low
-Close
-Volume

Technologies Used
-Python
-NumPy, Pandas
-Matplotlib
-Scikit-learn
-TensorFlow / Keras

Model Used
LSTM (Long Short-Term Memory)
LSTM is a type of Recurrent Neural Network (RNN) designed to learn patterns from sequential data, making it suitable for time series forecasting.
