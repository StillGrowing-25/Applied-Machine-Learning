This project develops a machine learning regression model to estimate house prices based on key features such as location, area, number of rooms, and available amenities. It involves systematic data preprocessing, exploratory data analysis, and feature engineering to improve model performance.

Multiple algorithms, including Linear Regression and Random Forest, are implemented and evaluated using standard metrics like Mean Squared Error and R² score to identify the most accurate model.

Tech Stack

Python, Pandas, NumPy, Scikit-learn, Matplotlib

Objective

To demonstrate the application of machine learning techniques in predicting real estate prices and supporting data-driven decision-making.
