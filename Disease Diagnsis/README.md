Disease Diagnosis using Clustering Techniques

This project focuses on applying unsupervised machine learning (clustering techniques) to analyze medical imaging data, such as X-rays, for disease diagnosis. The objective is to group similar patterns in the data and identify potential abnormalities without relying on labeled outcomes.

The workflow includes image preprocessing, feature extraction, and the application of clustering algorithms such as K-Means and Hierarchical Clustering to detect underlying structures in the data. The results are analyzed to assist in identifying patterns associated with different medical conditions.

Tech Stack
Python, NumPy, Pandas, Scikit-learn, OpenCV, Matplotlib

Objective
To demonstrate the use of clustering techniques in medical data analysis for discovering hidden patterns and supporting early disease diagnosis.
