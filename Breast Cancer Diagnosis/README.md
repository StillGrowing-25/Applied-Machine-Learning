Breast Cancer Diagnosis

This project focuses on developing a machine learning-based classification model to assist in the early detection of breast cancer using medical imaging data, such as mammograms. The model analyzes extracted features from images to classify cases as benign or malignant.

The workflow includes data preprocessing, feature extraction, and training classification algorithms such as Support Vector Machines, Logistic Regression, and Random Forest. Model performance is evaluated using metrics like accuracy, precision, recall, and ROC-AUC to ensure reliable diagnostic support.

Tech Stack
Python, NumPy, Pandas, Scikit-learn, Matplotlib

Objective
To demonstrate the application of machine learning techniques in medical diagnosis and support early, data-driven detection of breast cancer.
